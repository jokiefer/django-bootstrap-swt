======================================================================================================
django-bootstrap-swt - An app to simplifies the task of building HTML pages with bootstrap components.
======================================================================================================

Its features include:

- Permission checking before render the component.
- Provides auto id generating for bootstrap components.
- Includes the following bootstrap components:
  - Tooltip
  - Progressbar
  - Badge
  - Link
  - Button
  - Modal
  - Accordion
  - Card
  - Dropdown
  - more is coming soon...
- Javascript extensions to fetch content from url for Modal and Accordion component on opening/collapsing.

About the app:

- `Available on pypi <https://pypi.python.org/pypi/django-bootstrap-swt>`_
- `Documentation on readthedocs.org <https://django-bootstrap-swt.readthedocs.io/en/latest/>`_
- `Bug tracker <http://github.com/jokiefer/django-bootstrap-swt/issues>`_
